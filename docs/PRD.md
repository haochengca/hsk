# HSK 学习与默写系统 PRD（当前实现版）

## 1. 产品目标
- 面向 HSK1-6 的汉字/词汇学习与默写平台。
- 核心闭环：学习 -> 手写默写 -> 智能判定 -> 错题回流 -> 管理复判 -> 统计追踪。
- 目标是提高长期记忆效果与错题纠正效率。

## 2. 用户与权限
- 登录页统一入口；注册时需选择“父母/孩子”角色。
- 父母：可注册并登录。
- 孩子：可注册并登录，注册时需关联已存在的父母账号。
- 管理员：使用管理员账号登录。
- 孩子菜单：学习、写字、默写测验、错题本、我的记录。
- 父母菜单：默写判定审核、题本管理（不显示用户管理、学习项管理）。
- 管理员菜单：用户管理、题本管理、学习项管理（不显示默写判定审核）。
- 父母具备接近管理员的审核权限：可使用“默写判定审核”和“题本管理”，但不可使用“用户管理”“学习项管理”。

## 3. 功能需求

### 3.1 学习模块
- 支持按类型（汉字/词汇）与 HSK 等级筛选。
- 支持搜索、分页、勾选、多选。
- 学习列表取消“操作列”。
- “全部学习项列表”卡片需提供独立筛选区，与顶部学习卡筛选分离。
- “全部学习项列表”筛选项包括：
  - HSK 等级
  - 列表类型（汉字+词汇 / 仅汉字 / 仅词汇）
  - 默写次数
  - 准确率
- “全部学习项列表”标题单独一行居中显示，筛选项位于第二行并居中排列。
- 点击学习列表中的“汉字内容单元格”可直接进入写字页并开始笔画演示。
- 学习列表勾选结果会同步到写字页“汉字演示列表”数据源。

### 3.2 写字模块
- 写字页不提供可见“选择汉字”下拉（内部仍保留状态控件）。
- 页面主区两栏：左侧“临摹目标（含演示+手写）”，右侧“汉字演示列表”，顶部对齐。
- 手写框与演示框在同一卡片内，顺序为：
  - 临摹目标字
  - 演示笔画/重播
  - 笔画演示区域
  - 手写练习画布
  - 撤销一笔/清空/智能判定
- 取消“没写好”“写对了”按钮，仅保留“智能判定”。
- 写字页支持：
  - 上一个汉字 / 下一个汉字（位于汉字演示列表区）
  - 汉字演示列表点击汉字直接演示
  - 多选并批量演示，支持停止批量演示

### 3.3 默写测验模块
- 配置项：
  - 默写类型（汉字/词汇）
  - HSK 等级
  - 默写字数
  - 错题混入比例
  - 默写前预览（预览/不预览）
- 按钮状态：
  - 默认：开始默写可用，重新开始默写禁用
  - 进行中：开始默写禁用，重新开始默写可用
  - 禁用态按钮变灰、不可点击、鼠标 `not-allowed`
- 默写前预览规则：
  - 选“预览”时，开始默写前一次性展示本轮所有要默写的字/词
  - 展示总时长 = 本轮总字数 × 0.5 秒
  - 预览结束后进入正式默写，正式默写阶段不再预览
  - 选“不预览”时直接进入正式默写
- 播报顺序：字/词本身 -> 词组1 -> 词组2。
- 界面展示：拼音 + 英语解释。
- 点击“显示答案”会将当前题加入错题本（去重）。

### 3.4 手写判定与识别
- 判定为“写完后整体判定”，不按笔顺强制。
- 默写判定采用三引擎融合：
  - 轮廓重合引擎
  - 行列投影引擎
  - 网格结构引擎
- 融合得分用于通过/不通过判定；判定采用字复杂度分层阈值（simple/medium/complex）。
- 引入“不确定区间”：
  - 首次命中不确定区间时提示重写一次，不直接判错
  - 第二次仍不确定按失败处理（`uncertain_exhausted`）
  - 词汇题仅重写不确定字，已稳定字不重复书写
- 引入轻量在线学习（个性化 ML）能力：
  - 提取用户手写特征向量并在判定正确后增量更新个人原型
  - 个人原型参与后续判定融合，提高潦草但稳定个人写法的识别率
- 个性化原型更新门禁：仅最终通过且高置信（`decisionScore >= passThreshold + 0.02`）时更新。
- 前台不展示“识别准确度”数值，仅显示正确/错误。

### 3.5 词汇默写
- 词汇按字数生成多个手写窗口。
- 每个字独立判定，整词结果由逐字结果汇总（全对才算整词正确）。
- 每字手写图像需保存，用于记录和管理员复判。

### 3.6 错题本
- 错题本为独立页面。
- 错题列表仅显示字/词本身，不显示额外释义字段。
- 取消“清空错题本”按钮。
- 支持按 HSK 等级筛选按钮（全部、HSK1..n 动态生成）。
- 支持设置“错题默写字数”（5/10/20/全部）。
- 点击“错题默写”时，按“当前等级筛选 + 字数配置”出题。
- 错题移除规则：
  - 汉字：判对即移除
  - 词汇：写对累计次数 > 5（即第 6 次写对）才移除；再次写错重置累计

### 3.7 积分与奖励
- 每次默写正确 +1 分。
- 记录总积分、周积分、周正确次数。
- 周积分按周重置。
- 积分展示位于页面顶部。
- 加分动画策略：
  - 不在每道题结束时播放
  - 仅在“整轮默写全部完成”后播放一次全屏庆祝动画
  - 动画文案显示本轮累计得分（例如“本轮默写 +8 分”）

### 3.8 记录与统计
- 父母/孩子账号保存全部默写记录。
- 统计项包括：
  - 总次数、总正确率、正确/错误次数
  - 汉字正确率、词汇正确率
  - 最近7天/30天次数与正确率
  - 最近一次默写时间

### 3.9 默写判定审核
- 父母可查看所有用户提交、系统判定、手写图像。
- 管理员可查看全部用户列表（含角色与亲子关联信息）。
- 管理员可删除父母/孩子账号（管理员账号不可删除），删除后需同步清理该账号数据并维护亲子关联。
- 汉字提交：可改判正确/错误。
- 词汇提交：逐字改判后保存本词判定。
- 父母复判后联动更新用户错题本与积分。
- 复判完成后弹窗反馈是否已更新错题本/积分。

### 3.10 学习项管理
- 管理员可进入“全部学习项列表”查看所有汉字与词汇学习项。
- 支持按类型与 HSK 等级筛选学习项。
- 支持分页浏览学习项。
- 每个学习项支持维护：
  - 拼音
  - 提示词1
  - 提示词2
- 提示词默认值由前端根据词组映射自动生成；管理员保存后，以后台覆盖值为准。
- 覆盖值应支持批量保存与单项保存。
- 学习项覆盖值保存后，应同时影响默写播报与管理员列表展示。

### 3.11 默写会话保存策略
- 默写采用草稿会话机制：
  - 进行中：进度/错题/积分/提交先暂存
  - 全部完成：一次性提交
  - 中途取消（切菜单、退出、重开、改配置）：回滚草稿，数据不保存

### 3.12 抽题算法
- 默认抽题采用记忆曲线加权抽样：
  - 参考历史正确率、复习间隔、到期程度、最近错题状态
  - 错题混入比例在汉字默写场景继续生效
- 错题专项默写按错题筛选池抽样。

### 3.13 多 Agent 协作编排
- 系统内置多 Agent 分工能力，用于复杂任务拆解与协作执行。
- Agent 角色至少包括：
  - 任务分发 Agent（任务分类、流程选择）
  - 数据 Agent（样本与数据质量）
  - 模型 Agent（识别策略与阈值）
  - 评估 Agent（指标评估与风险门禁）
  - 产品 Agent（需求与验收对齐）
  - 交付 Agent（发布收口与复盘）
- 新任务进入时需执行：
  - 任务意图分类（如手写识别优化/功能交付/稳定性）
  - 按意图加载协作流水线
  - 与进行中任务比对，自动生成跨任务上下文交接动作
- 协作编排需支持命令行触发与结果可视化输出（文本或 JSON）。

## 4. 数据与技术约束
- 前后端分离，业务数据服务端持久化。
- 用户与记录数据存储在远程服务器。
- 响应式布局，桌面与移动端可用。
- 手写判定结果可追溯（含词汇逐字图像）。
- HSK 基础数据以 `data/hsk_source/L1.txt` 到 `L6.txt` 为唯一可机读源。
- 汉字数据需由源词汇文件自动生成，输出：
  - `data/hsk_chars_1_6.json`
  - `data/hsk_chars_1_6.js`
- 词汇数据需由源词汇文件自动生成，输出：
  - `data/hsk_words_1_6.json`
  - `data/hsk_words_1_6.js`
- 词汇数据生成规则：
  - 仅保留多字词
  - 单个汉字不得进入词汇表
- 汉字数据生成规则：
  - 以词汇源中首次出现的等级作为该汉字等级
  - 若存在单字词条，优先使用单字词条拼音与释义
  - 若无单字词条，可回退为“来自词汇”的释义
- 浏览器端学习页必须优先使用外部生成的数据脚本，不依赖手写维护的内嵌大数组。

## 5. 验收标准
- 父母/孩子从注册到学习、写字、默写、错题复习、记录统计全链路可用。
- 写字页支持汉字点击演示、批量演示、上下个切换。
- 学习页“全部学习项列表”支持独立 HSK 筛选，且不影响顶部学习卡的筛选状态。
- 学习页“全部学习项列表”标题与筛选区按“两行布局”展示：
  - 第一行标题居中
  - 第二行筛选项居中
- 默写预览符合：
  - 预览/不预览二选一
  - 预览为整轮统一展示
  - 总时长 = 字数 × 0.5 秒
- 积分动画符合：
  - 仅整轮结束后触发一次
  - 显示本轮累计加分
- 手写识别符合：
  - 三引擎融合判定生效
  - 个性化在线学习参与后续判定
- 多 Agent 协作符合：
  - 新任务可自动匹配协作流程
  - 存在进行中任务时可生成跨任务协作动作
- 错题本支持 HSK 分类与默写字数设置。
- 词汇错题满足“写对超过5次才移除”规则。
- 父母逐字复判可正确回写并联动错题/积分。
- 管理员修改学习项的拼音、提示词1、提示词2后，刷新页面仍能读取到保存结果。
